## CRA - React(js)
npx create-react-app <project-name>  
cd <project-name>  
yarn  
yarn dev  

## CRA - React(ts)
npx create-react-app <project-name> --template typescript  
cd <project-name>  
yarn  
yarn dev  

## Intall Storybook (6.x.x)
cd <project-name>  
npx sb init  

automatically detects project type and install everything needed
